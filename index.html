What is Duolingo?

Duolingo is the world's most popular language-learning platform. It's designed to feel like a game, making learning fun, engaging, and effective.

Evaluation Methods Used

1. Cognitive Walkthrough

Step 1: Selecting Course

The user easily selected “Korean” and tapped “Start Lesson.” The interface was intuitive. ✅ No confusion.

Step 2: Learning “ㄱ”

The app presented the character with its sound, but lacked a phonetic spelling (e.g., g/k), which could confuse beginners.

Step 3: Matching Character

Matching the character was straightforward, but repeated mistakes didn't offer any extra help or hints.

What is a Heuristic Evaluation?

A Heuristic Evaluation checks if the app follows common usability rules or 'heuristics'.

This report used Jakob Nielsen's 10 Heuristics, including:

  • Visibility of system status: Good feedback after such action (like sounds or animations)
  • Match between system and real world: Casual language fits users well
  • User Control and Freedom: No clear 'back' button during lessons
  • Consistency and standards: Icons are consistent but sometimes confusing (e.g., gems or hearts)

Suggestions for Improvement

  • Add phonetic spellings and clearer pronunciation guidance in Hangul lessons.
  • Show a “Lesson Completed” message or checkmark to confirm saved progress.
  • Provide brief explanations or hints after a user makes a mistake.

Summary of Usability Issues

Issue Affected Screen Problem Description
Lack of Phonetic Guide Hangul Lesson Users struggle with correct pronunciation without a clear phonetic reference (e.g., g/k).
Unclear Lesson Completion End of Lesson Users are unsure if their progress is saved, as there is no explicit confirmation message.
No Error Explanation Incorrect Answer Users don't learn from their mistakes because the app only shows the correct answer without hints.
